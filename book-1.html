    What’s inside

    This book is a hands-on, project-based guide designed to help you quickly get started with Laravel API development. You’ll build real application and learn how to integrate essential tools used in modern backend development.

    • Introduction to RESTfull API.
    • Introduction to Laravel.
    • Implement authentication, validation, routing, webhook etc.
    • Test endpoints with EchoAPI, and MailHog.
    • Deploy seamlessly using CapRover
    • Document your API with Scramble.
    • Build a Payment Processor API.
    • And lot more.
